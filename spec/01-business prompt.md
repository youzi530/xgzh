# Core Requirements (具体功能需求详解)

请根据以下 6 大核心模块及额外完善建议，帮我进行深度的产品设计与架构规划：

### 1. 新股 AI Agent（核心壁垒：低成本、高准确度、防幻觉）
- **输入输出**：输入新股代码/名称及申购信息（发行价、PE、招股书等），输出客观的基本面分析、风险点、多空情绪、预期上市表现及同业对比。支持多轮追问。
- **技术落地约束**：必须低成本。优先推荐当前最高性价比模型（如 DeepSeek-V3/R1、GLM-4-Flash，通过硅基流动等平台接入）。必须结合 RAG（检索增强生成）接入真实金融数据源，支持 Tool Use（工具调用）查询实时数据。
- **合规红线**：Agent 必须保持绝对中立，禁止给出“建议满仓/强烈建议申购”等绝对投资建议，必须添加免责声明。

### 2. 新股文章分析与知识汇总（港股 + A股）
- **功能设计**：区分港股/A股两个 Tab。自动检索全网主流公众号、媒体对特定新股的分析。列表需展示：标题、来源、摘要、情感倾向（看多/看空）。
- **AI 提效**：支持一键生成“多方观点总结 (TL;DR)”。
- **数据来源**：建议合法合规的网页爬取或搜索 API（如搜狗微信、雪球、东方财富等），展示时需标注来源。

### 3. CRS 报税全流程指导（向导式教学 + 模板工具）
- **功能设计**：将枯燥流程改造为 Step-by-Step 交互问卷（税务身份判定 → 填表向导 → 申报指南 → 个税辅助）。
- **内容提供**：提供标准文件模板下载（自我声明表等），支持新加坡、香港场景。基于 OECD、中国/香港/新加坡税务局公开指引。
- **合规红线**：强提示“不构成税务与法律建议，风险自担”。

### 4. 各大券商信息对比（个人福利与 CPA/CPS 变现核心）
- **功能设计**：港A股主流券商关键信息横向对比表（维度涵盖：佣金、平台费、融资利率、撤单费、打新活动等）。支持筛选与排序。
- **商业化闭环**：每家券商提供带有专属邀请码的官方注册/开户链接（追踪转化返佣）。

### 5. 历史新股中签数据 + AI 详细分析（数据资产库）
- **功能设计**：查询历史新股数据（中签率、首日表现、破发统计等）。附带 AI 统计分析（如：不同市值/保荐人的打新成功率规律）。
- **数据源与可视化 (Spike要求)**：请给出低成本数据抓取方案（如 A股推荐 AKShare/Tushare；港股推荐 Futu API/港交所公开数据）。前端需建议合适的数据可视化图表（如散点图、雷达图、K线）。

### 6. 个人中心与 VIP 会员体系
- **基础功能**：头像昵称、系统/打新/CRS提醒设置。
- **会员权益设计**：基于上述 1-5 点，设计合理的免费 vs 付费墙（例如：高级 Agent 无限对话、深层历史数据、无限模板下载、去广告等）。

### 7. 额外完善建议（体验与扩展）
- **首页设计**：今日/即将新股卡片 + AI 诊断快捷入口 + 打新日历。
- **UI/UX 原则**：专业简洁、数据可视化强、支持暗黑模式（Dark Mode）。
- **未来扩展**：美股 IPO、打新策略回测。

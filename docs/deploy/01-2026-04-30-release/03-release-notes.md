### C.2 关键架构能力

- **多数据源 dispatcher**: 7 (token 配后 9) 源, token-gated 灰度. 见 `spec/24` Sprint 8.
- **跨端 SSE 流式**: H5 fetch + AbortController / mp-weixin enableChunked + task.abort. 见 `apps/mp/utils/sse.ts`.
- **轻量 markdown parser**: 自实现 ~245 行, 跨端 `<view>+<text>`, 不走 `v-html` / `rich-text`. 见 `utils/markdown.ts`.
- **打字机节流**: 16ms 帧合并 SSE delta (H5 rAF / mp setTimeout). 见 `utils/typewriter.ts`.
- **PIPL 合规**: PII redact 在 Sentry breadcrumb (OPS-S5-001) + DB row level (BE-S5-002 phone redacted to last 4 + wechat_openid hashed in audit log).
- **三级内容审核**: 客户端简化版 + BE Tier 1/2 + 私域引流 + 隐私数字串. 见 `app/services/content_moderation.py` + Sprint 6.6 v3 升级.

# 新股估值速查 — PE / PB / PS 怎么看

打新要不要申购，**最简单的判断维度就是估值**。新手只要会看以下 3 个比率，就能筛掉 80% 的"明显贵"标的。

## 1. PE（市盈率） — 普通公司必看

公式：**PE = 股价 / 每股收益（EPS）= 总市值 / 净利润**

含义：**回本周期**。PE = 20 表示理论上 20 年回本。

### 1.1 怎么用

新股的招股书会披露 **发行 PE**（也叫 IPO 定价 PE）。

对比维度：
- **同行业上市公司平均 PE**（招股书一定会列）
- **历史上市新股 PE 区间**

> 比如恒生科技板块平均 PE 25 倍，新股发行 PE 30 倍 → 偏贵。

### 1.2 不适用场景

- **亏损公司**（PE 是负数或无穷大），用 PS 替代
- **银行 / 保险 / 房地产**（资产驱动），用 PB 更准

## 2. PB（市净率） — 重资产 / 金融公司看

公式：**PB = 股价 / 每股净资产 = 总市值 / 股东权益**

含义：相对账面价值的溢价倍数。PB = 1 表示按账面价买。

### 2.1 怎么用

- 银行 / 保险 / 地产，PB 通常 0.5-1.5
- 制造业，PB 通常 1-3
- 互联网 / 软件，PB 通常 5-15（轻资产，账面值少）

> 新股 PB > 5 的传统行业要警惕。

## 3. PS（市销率） — 亏损公司 / 高增长公司看

公式：**PS = 总市值 / 营业收入**

含义：回本周期的"营收版"。

### 3.1 怎么用

适合的公司类型：
- 高速增长但暂时亏损的科技公司（生物医药、SaaS、AI）
- 周期性公司（PE 失真时）

### 3.2 行业基准

| 行业 | 合理 PS 区间 |
|------|------|
| SaaS 软件 | 8-15 |
| 生物医药（未盈利） | 10-30 |
| 传统制造 | 0.5-2 |
| 金融 | 1-3 |

> PS 单独看意义不大，必须配合**营收增速**。营收增速 50%+ 的公司，PS 15 也不算贵。

---

## 综合速查

| 行业类型 | 主用估值 | 辅助维度 |
|------|------|------|
| 制造业、消费 | PE | 营收增速 |
| 银行、保险、地产 | PB | ROE |
| 科技、互联网 | PE 或 PS | 营收增速、毛利率 |
| 生物医药 | PS | 在研管线 |

## 新手避坑

1. **看招股 PE 是否高于同业**（招股书有同业对比表）
2. **不要只看 PE，搭配增速**（高增速可消化高估值）
3. **看历史新股**：同板块过去 6 个月新股表现，是最好的"市场体温计"

> 本文档不构成投资建议，估值仅是参考之一，不能替代深度基本面分析。
